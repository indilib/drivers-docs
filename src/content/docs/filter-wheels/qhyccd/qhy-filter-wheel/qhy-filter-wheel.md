INDI for Java Driver (fully compatible with the standard INDI library) for the QHY Filter Wheel (http://qhyccd.com/en/left/qhy-colorwheel/) using the RS-232 interface. You can check all the details about the driver and a small tutorial here: http://indilib.org/develop/indiforjava/qhy-filter-wheel-driver.html
